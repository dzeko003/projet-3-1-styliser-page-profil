# PROJET 3.1 : Styliser la Page de Profil

## Description

Ce projet a été réalisé dans le cadre de la formation **Akieni Academy – Phase 2**, **Semaine 3 : CSS - Box Model & Sélecteurs**.

L'objectif est de reprendre la page de profil réalisée lors du **Projet 2.1** et de l'améliorer visuellement grâce à CSS. Ce projet permet de découvrir les bases de la mise en forme d'une page web en utilisant des feuilles de style externes, le modèle de boîte (Box Model), les sélecteurs CSS et les polices Google Fonts.

---

## Objectifs pédagogiques

- Lier une feuille de style CSS externe à une page HTML.
- Comprendre le fonctionnement de la cascade CSS.
- Maîtriser le Box Model (`margin`, `padding`, `border`).
- Utiliser différents types de sélecteurs CSS.
- Améliorer la présentation d'une page web grâce à des styles cohérents.

---

## Fonctionnalités

La page de profil est enrichie avec les éléments suivants :

### Mise en forme

- Couleurs de fond
- Couleurs de texte
- Hiérarchie visuelle des titres
- Mise en page claire et agréable

### Typographie

- Intégration de Google Fonts
- Utilisation de plusieurs familles de polices
- Amélioration de la lisibilité

### Espacements

- Marges (`margin`)
- Espacements internes (`padding`)
- Respect des bonnes pratiques de mise en page

### Bordures

- Bordures simples sur différents éléments
- Mise en valeur de certaines sections

### Application du Box Model

Tous les principaux éléments de la page utilisent le modèle de boîte afin de mieux comprendre son fonctionnement.

## Critères de validation

Le projet respecte les exigences suivantes :

- Feuille de style CSS externe.
- Utilisation d'au moins 10 sélecteurs CSS différents.
- Application correcte du Box Model.
- Page lisible, cohérente et attrayante.
- Intégration d'au moins deux Google Fonts.
